Homework 2 Homework 2 aims to evaluate our skills in HTML5, CSS, and JavaScript through a multi-part assignment. The main page serves as a landing hub, featuring hyperlinks to Part 1, Part 2, and an optional Extra Credit section, as well as a link to return to the Homework 1 main page. In Part 1, we created a webpage that utilizes JavaScript to display styled text. Part 2 involves creating a webpage with a form that accepts three integers from the user; upon submission, various calculations are performed and displayed. Grading will consider script functionality, user experience, and code validity.
