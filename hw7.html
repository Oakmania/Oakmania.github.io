Homework 7 - Object Model and Cookies

  • Part 1

    Tasked students to create an interactive webpage where users can dynamically alter the appearance of a text area. This is achieved through a form that allows users to choose different styles, such as background color, text formatting (bold, italic, underline), and font size. As users select these styles, they are immediately applied to the text area, demonstrating real-time changes. This exercise showcases the use of HTML elements, CSS for styling, and JavaScript for dynamic interaction and immediate application of styles.

  • Part 2

    The goal is to develop a webpage that utilizes cookies to personalize the user experience. This is achieved by saving two pairs of cookies: one for the user's name and another for their username. On a user's first visit, they are prompted to enter this information. Upon subsequent visits, the webpage recognizes the user through these cookies and displays a personalized welcome back message. This exercise illustrates cookie handling in web development, providing a basic understanding of storing and retrieving user data to enhance interactivity and user experience on a website.
