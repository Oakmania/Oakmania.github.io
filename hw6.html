Homework 6 - Objects

  • Part 1

    This assignment focuses on developing a web page that includes a form for processing and displaying calculations based on user-inputted decimal numbers with at least four decimal places. The key objectives are to design a user-friendly interface, implement form validation in JavaScript to ensure the number meets specified criteria, and perform various mathematical operations.

  • Part 2

    Tasked with creating a web page where users can input a lengthy text and a single character to find its frequency within the text. The page should feature a form with a textarea for the text, an input box limited to one character, and an output area or window to display results. The task emphasizes implementing case-insensitive search functionality and handling different outcomes, displaying the character count directly on the page or showing a message in a new window if the character is not found.

  • Part 3

    In this assignment, the objective is to create a web page that allows users to enter a phone number in a specified format and then separates and displays the number's components (area code, first three digits, last four digits) in distinct fields. The task involves creating a user-friendly interface with input validation using JavaScript and regular expressions, ensuring users adhere to the required phone number format. It also aims to enhance skills in JavaScript for string manipulation and dynamic content presentation.
